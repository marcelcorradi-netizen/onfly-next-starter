# Tokens Input

Cole aqui os arquivos JSON exportados do Figma antes de rodar a conversão.

## Formato Plugin (preferido)

```
tokens-input/
├── manifest.json
└── foundations/
    ├── Brand.Value.json
    ├── Foundation.Value.json
    ├── Typography.Value.json
    ├── Theme.Light.json
    ├── Effects.Mode 1.json
    ├── Layout.Desktop.json
    ├── Layout.Mobile.json
    ├── Grid.Desktop.json
    ├── Grid.Mobile.json
    ├── text.styles.tokens.json
    ├── effect.styles.tokens.json
    └── grid.styles.tokens.json
```

## Formato Figma Native (fallback)

Se exportar direto do Figma (sem plugin), cole os ZIPs por collection aqui na raiz de `tokens-input/`.

---

Depois de colar os arquivos, invoque a skill `design-system-specialist` e descreva o que quer fazer.
